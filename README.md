# FRONTEND-PROJETO-WEB2

## ShopPobre - Frontend

### Descrição
Interface web do sistema **ShopPobre**, conectada ao backend para oferecer funcionalidades de e-commerce como login, listagem de produtos, carrinho e painel administrativo.

---

### Funcionalidades
- Login e cadastro de usuários (JWT)
- Listagem e busca de produtos
- Carrinho de compras
- Finalização de pedidos
- Histórico de compras
- Painel admin (gerenciamento de produtos e categorias)
